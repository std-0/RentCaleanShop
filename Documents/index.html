Product Categories Manager

Categories Manager is the tool where the product categories will be declared. This tool will allow the admin to add more categories as he wants. The admin can also add unlimited child categories for a parent category and able to delete and restore them.

Product Brands Manager

Product Brands Manager is the tool for adding product brands. This allows the admin to add the brands upload the brand logo, and can delete them as well. Also, the admin can make a brand a featured brand to show it on the store-front page.

Product Attribute Type Manager

A product may have several variations like sizes, colors, etc. From the Product Attribute Types manager, the admin can declare the attributes needed for all of his products.

Product Manager

Here you will get the list of all products. You can add a new product by clicking on the Add New button on the top-right corner of the screen. You may also edit, add all the variants, manage the inventory, delete and restore the product.

Product Reviews Manager

In this section, the admin can see the reviews of the product written by the users. And also able to delete or restore the reviews

Promotion

Promotion Code Manager is the tool where the admin can add discount type, percentage, offer, offer time limitations, and offer price limitations.

Shipping Method Manager

In this section, the admin can select the product shipping method and can delete as well.
